# Rules — Правила проекта

Общие правила кода. Используются всеми агентами.
Каждый агент читает только нужный файл, только когда он нужен.

---

## Файлы и когда читать

| Файл | Читать когда |
|------|-------------|
| `typescript.md` | Работаешь с типами, интерфейсами, generics, any |
| `components.md` | Создаёшь или рефакторишь React компонент (.tsx) |
| `styles.md` | Создаёшь или рефакторишь CSS/стили |
| `naming.md` | Называешь файл, папку, переменную, функцию |
| `utilities.md` | Упрощаешь логику, работаешь с массивами/датами/объектами |

---

## Для агентов

Перед началом работы — проверь что папка существует:

```bash
ls agents/rules/
```

Если папки нет или файл не найден — спроси у пользователя путь к правилам проекта.
